# 🍗 NutriBot — Telegram бот питания

Персональный план питания на 7 дней с простыми рецептами из курицы и мяса.
Бот рассчитывает калории по формуле Миффлина-Сан Жеора и масштабирует граммовку под каждого пользователя.

---

```
nutribot/
├── bot.py           # Основной файл бота

```

## 🚀 Деплой: шаг за шагом

### Шаг 1 — Получи токен бота

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя (например: `NutriPlan Bot`)
4. Придумай username (например: `nutriplan_bot`)
5. BotFather пришлёт токен — скопируй его, он понадобится

---

### Шаг 2 — Залей код на GitHub

```bash
# 1. Создай новый репозиторий на github.com (кнопка New → назови nutribot → Create)

# 2. Открой терминал в папке проекта и выполни:
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ТВОЙusername/nutribot.git
git push -u origin main
```

---

### Шаг 3 — Деплой на Railway

1. Зайди на **railway.app** → войди через GitHub
2. Нажми **New Project → Deploy from GitHub repo**
3. Выбери репозиторий `nutribot`
4. Railway автоматически найдёт `Procfile`
5. Перейди в раздел **Variables** и добавь переменную:
   ```
   BOT_TOKEN = токен_от_BotFather
   ```
6. Нажми **Deploy** — бот запустится через ~1 минуту

---

### Шаг 4 — Проверь

Открой своего бота в Telegram и напиши `/start`

## 🧮 Как считаются калории

**Формула Миффлина-Сан Жеора:**
- Мужчины: `BMR = 10×вес + 6.25×рост − 5×возраст + 5`
- Женщины: `BMR = 10×вес + 6.25×рост − 5×возраст − 161`

**TDEE** = BMR × коэффициент активности

**Цель:**
- Похудение: TDEE − 500 ккал
- Поддержание: TDEE
- Набор массы: TDEE + 500 ккал

**Макросы:** Белки 30% / Углеводы 45% / Жиры 25%

## 🥩 Граммовка

В каждом рецепте показаны **два варианта** протеина:
- 🐔 **Курица** — больше граммов (менее калорийная)
- 🥩 **Мясо** (говядина/свинина) — меньше граммов (более калорийное)

Граммовка автоматически масштабируется под калорийную норму пользователя.
